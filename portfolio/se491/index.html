Holomon

Holomon is an adventure/exploration game for the Hololens based on Pokemon GO. In it you can catch and use creatures we are calling "holomon" to battle other people in-game. You wander around a building for a little bit, happen to spot a holomon, then you can approach it and use holomon you have already collected in order to try and capture it. You might also encounter a virtual "gym" where you can use the holomon you have captured in order to battle an AI, who also has a collection of holomon.

We are developing this game using Unity. The purpose of the project is to research how well different types of UI work well in the hololens, so we can apply it to everyday applications, while also gaining experience developing a game.
